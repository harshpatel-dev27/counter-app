# React Counter App

This is a simple counter application built using React.  
I created this project to practice React basics like state management, hooks, and component structure, and then deployed it on Vercel.

## About the Project

The app allows users to increase or decrease a counter value.  
It also supports step-based counting, reset functionality, and a theme toggle.  
While building this project, I focused on writing clean React code and understanding how a real React app is deployed.

---

## Features

- Increment and decrement counter
- Step selection (1, 5, 10)
- Prevents negative values
- Reset counter to zero
- Light / Dark mode toggle
- Updates browser tab title dynamically
- Simple and clean UI
